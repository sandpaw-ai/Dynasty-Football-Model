# v3.0 — Conference-Tier Mapping for Pre-2014 NCAA Football

**Status:** v3.0 PR 1
**Scope:** classification of FBS conference rows in the 2000-2013 season files into the existing `{P5, G5_top, G5, FCS}` schema used by the 2014+ cfbfastR corpus.

## Why this document exists

The existing `data/historical_ncaa_football/season_{2014..2025}.json` files are derived from `cfbfastR`, which tags every player-season with a `conference_tier` field. v3.0 extends that corpus back to 2000 using `sports-reference.com` data, which only exposes raw conference *names* — we have to do the tier classification ourselves.

The tier matters because v3.0 PR 3's similarity engine multiplies college fantasy production by a tier coefficient (`P5=1.0, G5_top=0.85, G5=0.75, FCS=0.65`) before comping players. Mis-tiering a conference biases an entire era of comps.

## The four tiers (recap from cfbfastR)

| Tier | Meaning |
|---|---|
| **P5** | Power conference / autonomy-granting (the "Power Five"). |
| **G5_top** | Strongest of the Group of Five — historically AAC + Mountain West. |
| **G5** | The other FBS conferences — C-USA, MAC, Sun Belt, FBS Independents. |
| **FCS** | Sub-FBS / FCS-level. Default for any conference we can't identify. |

## Year-by-year mapping (FBS conferences)

| Conference | 2000–2010 | 2011 | 2012 | 2013 |
|---|---|---|---|---|
| SEC | P5 | P5 | P5 | P5 |
| Big Ten | P5 | P5 | P5 | P5 |
| Big 12 | P5 | P5 | P5 | P5 |
| ACC | P5 | P5 | P5 | P5 |
| Pac-10 / Pac-12 | P5 (Pac-10) | P5 (Pac-12) | P5 | P5 |
| **Big East** | **P5** | **P5** | **P5** | G5_top |
| AAC (renamed from Big East football, 2013) | — | — | — | G5_top |
| Mountain West (MWC) | G5_top | G5_top | G5_top | G5_top |
| **WAC** | G5 → G5_top | G5_top | G5_top | G5 (football collapsed) |
| Conference USA (C-USA) | G5 | G5 | G5 | G5 |
| MAC | G5 | G5 | G5 | G5 |
| Sun Belt | G5 | G5 | G5 | G5 |
| FBS Independents (Notre Dame, BYU, Army, Navy, etc.) | G5 | G5 | G5 | G5 |

> *Independents:* Notre Dame is a borderline case — competes at a P5 level but isn't in a P5 conference. We follow cfbfastR's convention and tag all "Ind" rows G5. Notre Dame players who matter for comps are flagged by their NFL outcome via the bridge, not by tier inflation. The model can absorb this via name-level overrides in PR 3 if back-test demands.

## The Big East / AAC transition

This is the most consequential year-sensitive call. The Big East was a football power conference through the 2012 season. Major football members (Pittsburgh, Syracuse, West Virginia, Louisville, Rutgers, Cincinnati, UConn, etc.) generated genuine P5-tier production from 2000-2012. Notable NFL players whose college careers fall in this window — Larry Fitzgerald (Pitt, 2002-03), Geno Smith (West Virginia, 2009-12), Bryce Petty's predecessor era, etc. — would be unfairly demoted to G5 if we wholesale-G5'd the Big East.

After the 2012 season the football schools (with some additions) renamed to the American Athletic Conference (AAC). The AAC has *never* been a Power-Five conference, so it gets G5_top — same slot the Mountain West has held.

**Rule encoded in `sports_reference_cfb.classify_conference_tier`:**

```python
if conference in {"Big East", "BE"}:
    return "P5" if season <= 2012 else "G5_top"
```

## The WAC peak

The Western Athletic Conference was, for most of its history, a sleepy mid-major. But from roughly 2007 through 2012 — Boise State's BCS-buster era, Hawaii's 12-0 Sugar Bowl run, TCU briefly there — it punched well above its weight. cfbfastR doesn't directly mirror this, but the Phil case (Kellen Moore's 35-TD season at Boise State in 2010, etc.) is exactly the kind of production we *should* discount relative to a SEC equivalent, and bucketing the 2007-2012 WAC as G5_top (instead of plain G5) keeps it in the same neighborhood as the Mountain West it later joined.

The 2013 season is excluded from G5_top because the football conference effectively collapsed that year (most schools defected to MWC or C-USA; Idaho went FBS Independent).

```python
if conference == "WAC" and not (2007 <= season <= 2012):
    return "G5"
```

## What we don't try to encode

- **Individual-program strength.** Notre Dame plays a P5 schedule but is tagged G5 via Independents; back-test will reveal whether this hurts comp accuracy enough to warrant a per-team override. PR 5's "biggest misses" analysis is the right place to surface it.
- **Schedule-strength within tier.** SOS is its own corpus (PR 2). A SEC team that played 6 cupcakes and an SEC East champion is in P5 but should get a *lower* SOS multiplier than an SEC team that played the harder slate. Tier and SOS combine multiplicatively in the engine; this doc is just about tier.
- **Year-by-year membership shuffles within tiers.** Schools move conferences a lot (Texas → Big 12 → SEC; etc.). We follow whatever `team_name_abbr` + `conf_abbr` sports-reference emits for the season; that captures the right tier for the *team-season* we're scoring.

## How to update this

If a back-test miss case (PR 5) is clearly explained by a conference-tier error — e.g., "we tagged Larry Fitzgerald's 2003 Pitt as G5_top because we shipped the year-guard wrong" — fix the function in `sports_reference_cfb.py`, add a test in `test_sr_cfb_scraper.py`, and update the table above. The function is the source of truth; this doc explains the why.

The function is intentionally pure (no I/O, no state) so the v3.0 engine can call it on the fly for any (conference, season) tuple.
