# v2.4 PR 4 — Go-Live: Flag Flip + UI Badge + Methodology Update

**Status:** ready to merge
**Sequence:** Final PR in the v2.4 stack. Depends on PRs 1, 2, 3 (all merged).

## What this PR does

1. **Flips `USE_PRE1999_CORPUS` default from `False` → `True`.** Env-var unset
   now means pre-1999 ON. Set `USE_PRE1999_CORPUS=false` to opt back into the
   1999+-only path (used by tests that pin the legacy behaviour).

2. **Adds the ⏳ era badge** to comps on player pages whose snapshot season
   predates 1999. Hover title explains the 0.9× confidence haircut from PR 3.

3. **Updates the Known Limitations section** of the methodology page —
   removes the "Corpus starts in 1999" caveat that PRs 1-3 made obsolete.
   New text explicitly documents the 1980 floor + the haircut + pre-1980
   as out-of-scope for v2.4.

4. **Rebaselines one PR 2 test** that assumed unset-env = off. Two-line
   change to explicitly set `false` instead of unsetting.
